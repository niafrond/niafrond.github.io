📖 Règles rapides

3 manches sur les mêmes mots réunionnais — les équipes jouent à tour de rôle (30 s par tour) :

🗣️ Manche 1 : Parler librement
⛔ Interdit : dire le nom, épeler, traduire
✅ Autant de propositions qu'ils souhaitent

☝️ Manche 2 : Un seul mot
✅ Passer autorisé — une seule tentative par carte
❌ Mauvaise réponse → carte remise dans le jeu

🤐 Manche 3 : Mime + bruitages uniquement
✅ Mêmes règles que la manche 2 — passer autorisé
⛔ Aucun mot, aucune chanson fredonnée

🧮 1 carte trouvée = 1 point · Mémorisez les mots !
